Introduction

Welcome to the user manual for E-ticketing System. This document is intended to provide you with all the information you need to get started with and use the system effectively.

The E-ticketing user interface is designed to be intuitive and easy to use. Here's a brief overview of the different parts of the interface:

  • Landing Page
  • Reporting an Issue
  • Escalating an Issue
  • Suggesting an Idea
  • Assigning Tickets
  • Surveys

Using E-ticketing System

To use E-ticketing System please follow these steps:

Process 1: Landing Page

To access various functionalities of the E-ticketing System, start from the landing page:

  1. Click on "Report an issue" to raise a ticket.
  2. Click on "Suggest an idea" to propose new features or modules that can enhance the system's efficiency.
  3. Click on "Escalate an issue" if a previously raised ticket requires further support.
  4. Click on "Knowledge base" to learn more about the operationalization of the system.

Process 2: Reporting an Issue

To raise a new ticket, follow these steps:

  1. Navigate to the "RAISE A NEW TICKET" section.
  2. Fill in the required details:
    • Category: Select or search for the ticket category.
    • Priority: Choose the ticket priority level.
    • Ticket Title: Provide a concise title for the ticket.
    • Ticket Description: Describe the issue or request in detail.
  3. Click the "Submit" button to raise the ticket.

Process 3: Assigning a Ticket

To assign an existing ticket to an assignee, follow these steps:

  1. Access the "ASSIGN-TICKET" section.
  2. Complete the following fields:
    • Ticket Code: Enter the ticket code or search for the ticket.
    • Assignee: Enter the assignee's details.
    • Status: Update the status of the ticket.
    • From Date and To Date: Select the date range for the ticket.
  3. Click "Submit" to assign the ticket.

Process 4: Escalating a Ticket

To escalate an existing ticket for further support, follow these steps:

  1. Access the "ESCALATE TICKET" section on the dashboard
  2. Fill in the required details:
    • Category: Select or search for the ticket category.
    • Priority: Choose the escalated ticket priority level.
    • Ticket Title: Provide a concise title for the escalated ticket.
    • Escalate Description: Describe why the ticket needs to be escalated.
  3. Click the "Submit" button to escalate the ticket.
